# Prospect Signal AU website

Static HTML/CSS/JS website for Prospect Signal AU. No framework. Deployable to Netlify with publish directory `.`.

## Built files

- `index.html`
- `products.html`
- `mining.html`
- `methodology.html`
- `responsible-use.html`
- `product/*.html` generated for 202 products
- `data/products.json` contains 202 product records

Buy buttons use Gumroad now and include `data-stripe-url` for Stripe link swapping later.
